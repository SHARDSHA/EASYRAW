# EASYRAW
EASY RAW DEVELOPER WITH A SOUL

Written entirely using IA tools because I have no idea to code. Main goal of is to learn how to develop a python project and how to use github. Secondary goal is to have a very simple raw development tool that gives "that touch" to the pictures without getting lost in complex GUIs and workflows.
